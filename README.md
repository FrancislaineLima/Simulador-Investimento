# Simulador-Investimentos-Fundos-Imobili-rios
Simulador interativo de Fundos Imobiliários desenvolvido no Excel. Esta ferramenta oferece um dashboard intuitivo para projetar a evolução patrimonial e a renda passiva (dividendos) ao longo do tempo. Essencial para investidores que buscam simular cenários de aportes e rentabilidade para planejar sua independência financeira com dados precisos
